Info

The Introduction to Scientific Computing in Data Science with Python workshop is an introductory course aimed at those with no prior programming experience. In this workshop, you will learn about the fundamentals of Python. The workshop is hands-on and you will learn scientific computing skills for data analysis using NumPy and Pandas packages, as well as data visualisation with Matplotlib package. It is therefore best that you bring your own laptop and download the data science toolkit from Anaconda.

Software requirements

Participants are expected to download and install Anaconda BEFORE attending the workshop. You may do so by clicking Download on their website and choose the installer that is compatible with your computer system. If you have any queries or encounter any issues, please do not hesitate to contact me at marieanne@umk.edu.my.
